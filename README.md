
# Employee Record Management System (Java)
A console-based Java application to manage employee records using the Java Collections Framework.  
This project supports basic CRUD operations and demonstrates efficient data handling using HashMap.

# Features
1. Add new employee records
2. Update existing employee details
3. Delete employee records
4. Display all employee records
5. Menu-driven user interaction
6. Prevents duplicate employee IDs

# Technologies Used
1. Java (Core Java)
2. Java Collections Framework
     1. HashMap
     2. Map
3. Scanner (User Input)

# Concepts Covered
1. Object-Oriented Programming (OOP)
2. Encapsulation
3. toString() method
4. Collections (Map)
5. CRUD Operations
6. Menu-driven programs


# Project Structure
employee_record_management/Employee.java/EmployeeManagement.java


# How It Works
1. Employee ID is used as a unique key.
2. Employee objects are stored as values inside a HashMap.
3. HashMap ensures fast insertion, update, deletion, and retrieval.
4. `toString()` is overridden for meaningful output display.
